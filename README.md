$project
========

$project will give you the data of peoples 
from (https://swapi.co/)api in cards of material design.
   

Features
--------

- use material design
- filter cards by input search
- drag and drop cards  

Installation
------------

Install $project by running:

  - clone the project first
  -  cd 'project path'
   - ng serve --open
